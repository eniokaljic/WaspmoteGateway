WaspmoteGateway
===============
Gateway for connecting of Waspmote nodes via 802.15.4/ZigBee or TCP to OpenHAB.

To connect 802.15.4 / ZigBee modules to the gateway a serial adapter is needed (eg. USB-UART converter).
